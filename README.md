A sleek, modern, and high-performance digital stopwatch featuring a stunning neon-dark aesthetic. Designed for precision and visual appeal, this project blends minimal design with vibrant glowing accents to create a professional-grade timing tool.

## ✨ Features

- **🌌 Cyberpunk Aesthetic**: A deep dark theme complemented by glowing neon accents for a futuristic look.
- **🕒 Digital Precision**: A digital clock-style display (`#display`) using monospace fonts and glowing text to mimic real-world electronic timers.
- **⚡ Interactive Controls**: Smooth, responsive hover and active animations on all control buttons (`#Startbtn`, `#stoptbtn`, `#resetbtn`) for a tactile feel.
- **🎯 Perfect Balance**: Fully centered on-screen layout using a balanced container (`#container`) for a focused user experience.
- **✍️ Elegant Typography**: Integrated with **Google Fonts** to ensure a clean and professional typographic hierarchy.
- **📱 Fully Responsive**: Visually balanced across different screen sizes.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure for a robust foundation.
- **CSS3**: Advanced styling including Flexbox/Grid for centering, CSS3 animations, and neon box-shadow effects.
- **JavaScript (ES6+)**: High-precision timing logic for start, stop, and reset functionality.
- **Google Fonts**: Premium typography for the digital display.

## 🚀 How To Use

1. **Clone the repository** to your local machine.
2. **Open `index.html`** in any modern web browser.
3. **Control the timer**:
   - Click **Start** 🟢 to begin tracking time.
   - Click **Stop** 🔴 to pause the timer.
   - Click **Reset** 🔄 to clear the display back to `00:00:00:00`.
